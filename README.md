# Гоночный симулятор (Race Simulator)



## Описание проекта

Консольное приложение на C++, симулирующее гонки фэнтезийных транспортных средств.  

Программа позволяет создавать гонки различных типов (только наземный, только воздушный или смешанный транспорт), регистрировать участников и определять победителя на основе времени прохождения дистанции.



## Функциональность

- Выбор типа гонки:

- только для наземных ТС;

- только для воздушных ТС;

- для наземных и воздушных ТС.

- Задание длины дистанции (положительное число).

- Регистрация транспортных средств:

- каждое ТС может быть зарегистрировано только один раз;

- нельзя зарегистрировать ТС, не соответствующее типу гонки;

- минимум два участника для старта.

- Расчёт времени прохождения дистанции с учётом уникальных характеристик каждого ТС.

- Вывод результатов гонки (сортировка по возрастанию времени).

- Возможность провести несколько гонок подряд.



## Транспортные средства

Реализованы следующие виды ТС:



| Название               | Тип      | Особенности |

|------------------------|----------|-------------|

| Верблюд                | Наземный | Отдых: первый 5, последующие 8 |

| Верблюд-быстроход      | Наземный | Отдых: 5, 6.5, затем 8 |

| Кентавр                | Наземный | Отдых всегда 2 |

| Ботинки-вездеходы      | Наземный | Отдых: первый 10, последующие 5 |

| Ковёр-самолёт          | Воздушный | Коэффициент дистанции зависит от длины |

| Орёл                   | Воздушный | Коэффициент всегда 0.94 (сокращение на 6%) |

| Метла                  | Воздушный | Коэффициент = 1 - (целое число тысяч / 100) |



## Архитектура

Проект разделён на две части:



### RaceLibrary (динамическая библиотека DLL)

Содержит иерархию классов:

- `Vehicle` – абстрактный базовый класс.

- `GroundVehicle` – базовый класс для наземных ТС.

- `AirVehicle` – базовый класс для воздушных ТС.

- Конкретные классы: `Camel`, `FastCamel`, `Centaur`, `Boots`, `MagicCarpet`, `Eagle`, `Broom`.

- `Race` – класс для управления гонкой (регистрация, старт, результаты).



### RaceSimulator (исполняемый файл EXE)

Содержит логику взаимодействия с пользователем:

- меню выбора типа гонки;

- ввод дистанции;

- меню регистрации ТС;

- запуск гонки и вывод результатов.



Классы библиотеки **не выводят данные на консоль** – вся работа с вводом/выводом сосредоточена в `main.cpp`.
